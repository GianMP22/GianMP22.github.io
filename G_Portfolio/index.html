Cura Infill Examples:
Fusion 360

#3Dprint, #CAD, #GraphicDesign, #Work


While working in the 3D printing sections of my major's design lab (basically a mini makerspace), I found it hard to describe the different infills the Cura slicer offers without just showing the model sliced with each infill side by side. I wanted an easier way to show it so I made these infill examples by making a hexagon model - so that they fit together aesthetically - and setting the top layer size to 0 layers so it wouldn't print.
