Textured Arc Reactor:
Maya & Cinema 4D

#3D, #Renders, #SchoolProject


Problem Statement:

I had this great 3D model of an Iron Man Arc Reactor that I made in a 3D modeling class while learning Maya. I always wanted to post it but I never felt like it was good enough. The goal for this project was to use C4D's texturing and renderer to make this model look movie accurate.
Like the Mark 1 Arc Reactor from the Iron Man movie, I wanted this model to look like it was built in a cave from a box of scraps. So I wanted it to look used and worn down.


Inital Renders:

I went with 3 different styles for the Arc Reactor.
The first was an "off" version - which had no glow - in order to show off the work I put into the worn metal textures (shown below).

The second version was the "on" version with the classic blue glow.

Finally, the third version was an old worn down "on" version with a yellow glow.

Final Renders:

For the final iteration I indented the text around the ring of the Arc Reactor stand. Also, I forgot to check the "bloom" setting to make the glow look more realistic in the renderer so I fixed that in the final thing (shown below).

I also made a TIME magazine mockup just for fun.

And finally I made a mockup from Tony Stark's lab as the background with a screenshot that I got from the movie that I then extended down to make the table using photoshop's built-in AI.
