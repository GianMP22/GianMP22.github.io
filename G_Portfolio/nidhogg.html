Custom Nidhogg Game Controller:
Fusion 360 & Circuit Playground Express

#Prototype, #3Dprint, #CAD, #SchoolProject


Project Description:

The idea for this project was to utilize a microcontroller's ability to perform keypresses in order to redefine how the game is played. I chose Nidhogg because it has a relatively simple gameplay and my friends enjoy playing this game together as a 1v1.

Process:

I started by writing down all the possible actions a player can perform in-game.
